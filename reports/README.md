# reports/

This directory stores all **output artifacts** generated from the research pipeline: figures, tables, and narrative documents. These outputs may be referenced directly in the final report or paper.

---

## Directory Structure

```
reports/
├── figures/               # Saved plots and charts (PNG, PDF, SVG)
├── tables/                # Summary tables (CSV, LaTeX)
├── results_template.md    # Template for reporting final results
└── presentation_notes.md  # Notes for slidedeck / oral presentation
```

---

## File Conventions

### Figures (`figures/`)

- Format: **PNG** (screen/web) or **PDF** (paper/LaTeX).
- Resolution: minimum 150 DPI for screen; 300 DPI for publication.
- Naming: `{type}_{description}.{ext}`, e.g., `equity_curve_xgboost_daily.png`.
- Always include axis labels, titles, and legends.

### Tables (`tables/`)

- Format: **CSV** (primary) + optionally LaTeX (`.tex`) exported from analysis.
- Naming: `{description}.csv`, e.g., `model_comparison.csv`, `backtest_summary.csv`.
- Rows = models or configurations; columns = metrics.
